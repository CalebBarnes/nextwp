# Quickstart

Deploy a project in minutes with this quickstart guide. {{ className: 'lead' }}

## WordPress hosting

Connect to an existing WordPress instance, or create a new WordPress site using any hosting provider such as [WP Engine](https://wpengine.com/?utm_source=nextwp&utm_medium=docs&utm_campaign=nextwp-docs), [DigitalOcean](https://m.do.co/c/8c72c601817b), or [Kinsta](https://kinsta.com/?kaid=ZWZKLOAMLNDP).<br/>
We recommend [WP Engine](https://wpengine.com/?utm_source=nextwp&utm_medium=docs&utm_campaign=nextwp-docs) as they have a great developer experience, reliable hosting and amazing support.

## Pre-requisites

NextWP integrates with the [NextWP Toolkit](/packages/wordpress/nextwp-toolkit-plugin) plugin on your WordPress site to enable features like preview mode and to retrieve additional data not included by default in the WP REST API, such as ACF options pages.

Or, you can create a new project using the NextWP CLI tool.

To create a new NextWP project, you can use the `create-nextwp-app` CLI tool to generate a starter project. This tool will automatically set up a new project for you, including all the necessary dependencies and configuration files to get you started with NextWP.

<Note>
  The NextWP CLI tool is currently very new and experimental. If you run into
  any issues, please let us know.
</Note>

```bash {{ title: 'yarn' }}
yarn create nextwp-app
```

## Add to existing project

If you already have a Next.js project, you can install `@nextwp/core` and optionally `@nextwp/addons` using your preferred package manager.

<CodeGroup>

```bash {{ title: 'npm' }}
npm install @nextwp/core
```

```bash {{ title: 'pnpm' }}
pnpm add @nextwp/core
```

```bash {{ title: 'yarn' }}
yarn add @nextwp/core
```

</CodeGroup>

## Add Environment Variables

Properly configuring environment variables is crucial for enabling the NextWP to WordPress connection.

Generate an application password in WordPress. (Users > Your Profile > Application Passwords)<br/>
Make sure your `NEXT_PREVIEW_SECRET` and `REVALIDATE_SECRET_KEY` are also added in your [NextWP Toolkit](/packages/wordpress/nextwp-toolkit-plugin) plugin settings.
