## What is NextWP? {{ anchor: false }}

NextWP is a library built on the WP REST API and Next.js App Router that streamlines the process of building super fast Headless WordPress sites with Next.js.
It provides a set of tools and utilities to help you build interactive, dynamic, and performant sites with WordPress and Next.js. {{ className: 'lead' }}
