## Recommended plugins

The NextWP Headless WordPress theme will suggest and optionally auto-install the following plugins:

- [NextWP Toolkit](/packages/wordpress/nextwp-toolkit-plugin) (required): The NextWP Toolkit plugin provides the core functionality for NextWP, including [previews](/packages/nextwp/core/route-handlers#next-word-press-preview).
- [On-Demand Revalidation](https://wordpress.org/plugins/on-demand-revalidation/): Triggers the [revalidate endpoint](/packages/nextwp/core/route-handlers#revalidate) in Next.js when content is updated in WordPress.
- [Advanced Custom Fields](https://wordpress.org/plugins/advanced-custom-fields/): ACF is a powerful tool for creating custom fields in WordPress.
- [YoastSEO](https://wordpress.org/plugins/wordpress-seo/): YoastSEO is a powerful SEO plugin. `@nextwp/core` uses YoastSEO to [generate metadata](/packages/nextwp/core/next-app-functions#generate-metadata) for Next.js.
- [Custom Templates Creator](https://github.com/CalebBarnes/custom-templates-creator): A plugin to easily add page templates to WordPress without the need to create php files.
- [Gravity Forms](https://www.gravityforms.com/): Gravity Forms is a powerful form builder for WordPress.<br/>
  <Note>
    `@nextwp/addons` provides a [NextGravityForm
    component](/packages/nextwp/addons#next-gravity-form) to easily integrate
    Gravity Forms with Next.js.
  </Note>
